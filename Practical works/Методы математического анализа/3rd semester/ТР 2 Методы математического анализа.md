### Вариант 5

$$\begin{gather}
\sum_{n=1}^{\infty} \frac{(-1)^{n+1}n^{n}}{(2n+1)^{n}}
\end{gather}$$

$$\begin{gather}
|a_{n}| = \frac{n^{n}}{(2n+1)^{n}} \\
\lim_{ n \to \infty } \sqrt[n]{ |a_{n}| } = \lim_{ n \to \infty } \frac{n}{2n+1} = \frac{1}{2} - \text{ряд сх-ся абсолютно}
\end{gather}$$
