### Вариант 5

$$\begin{gather}
\sum_{n=1}^{\infty} e^{-(1-x\sqrt{ n })^{2}}, \ x \in [1,2]
\end{gather}$$

$$\begin{gather}
|a_{n}| = \frac{n^{n}}{(2n+1)^{n}} \\
\lim_{ n \to \infty } \sqrt[n]{ |a_{n}| } = \lim_{ n \to \infty } \frac{n}{2n+1} = \frac{1}{2} < 1 \implies \text{ряд сх-ся абсолютно}
\end{gather}$$
