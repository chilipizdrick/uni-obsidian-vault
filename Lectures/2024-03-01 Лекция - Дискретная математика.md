## Кольца многочленов

$$A[x] = \{ a_{0} + a_{1}x + a_{2}x^{2} + \dots + a_{n}x^{n} + \dots \}$$

## Операции в кольцах многочленов

$$(P+Q)(x) = (p_{0}+q_{0}) + (p_{1}+q_{1})x + \dots + (p_{n}+q_{n})x^{n}$$

$$P\cdot Q = \sum_{k=0}^{\infty} c_{k}t^{k}, \ c_{k} = a_{0}b_{k} + a_{i}b_{k-1} + \dots + a_{k}b_{0} = \sum_{i=0}^{k} a_{i}b_{k-i}$$

> [!note]
> Кольцо многочленов - это [[Абелева группа]] по сложению и [[Моноид|моноид]] по умножению.

\end{gather}$$
