## Types of infinitives' forms

|                          | Active                 | Passive                |
| ------------------------ | ---------------------- | ---------------------- |
| Simple                   | to + $V$               | to be + $V_{3}$        |
| Progressive (Continuous) | to be + $V$            | -                      |
| Perfect                  | to have + $V_{3}$      | to have been + $V_{3}$ |
| Perfect progressive      | to have been + $V_{3}$ | -                      |
