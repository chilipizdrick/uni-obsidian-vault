### 1) $\int R(x, \sqrt{ a^{2}-x^{2} }) \, dx$

> Применяем замены:
> $$x = a\sin t, \ x = \tanh t$$

Такой интеграл можно решить проще:

$$\begin{gather}
\int x\sqrt{ 4-x^{2} } \, dx = -\frac{1}{2}\int \sqrt{ 4-x^{2} } \, d(4-x^{2}) = \\
= -\frac{1}{2} \cdot \frac{2}{3}(4-x^{2})^{3/2} + C = -\frac{1}{3}(4-x^{2})^{3/2} + C
\end{gather}$$

А такой нельзя:
